概念

线性相关和线性无关

  1. 向量

『向量』这一概念被提出用以表示一组有序的数据（即数组）。n 个数构成的一个有序数组被称为一个 n 维『向量』。设 \(\boldsymbol{\alpha}=[a_1,a_2,\ldots,a_n]^{\text{T}}\)，则 \(\boldsymbol{\alpha}\) 是一个 n 维『列向量』，\(\boldsymbol{\alpha}^{\text{T}} = [a_1,a_2,\ldots,a_n]\) 是一个 n 维『行向量』。

  • 一般情况下，向量默认为列向量，使用列向量的转置来表示行向量。
  • n 维列向量可看作 n 行 1 列的矩阵，n 维行向量可看作 1 行 n 列的矩阵。
  1. 线性组合 & 线性表示

设有 n 个 m 维向量 \(\boldsymbol{\alpha}_1,\boldsymbol{\alpha}_2,\ldots,\boldsymbol{\alpha}_n\) 与 n 个数 \(k_1,k_2,\ldots,k_n\)，则

\[ k_1\boldsymbol{\boldsymbol{\alpha}}_1 + k_2\boldsymbol{\alpha}_2 +\cdots+ k_n\boldsymbol{\alpha}_n \]

称为向量组 \(\boldsymbol{\alpha}_1,\boldsymbol{\alpha}_2,\ldots,\boldsymbol{\alpha}_n\) 的『线性组合』。若向量 \(\boldsymbol{\beta}\) 能表示为向量组 \(\boldsymbol{\alpha}_1,\boldsymbol{\alpha}_2,\ldots,\boldsymbol{\alpha}_n\) 的线性组合，则称向量 \(\boldsymbol{\beta}\) 能被向量组 \(\boldsymbol{\alpha}_1,\boldsymbol{\alpha}_2,\ldots,\boldsymbol{\alpha}_n\) 『线性表示』。

线性组合可以用矩阵乘列向量的方式表示：

\[ k_1 \boldsymbol{\alpha}_1 + k_2\boldsymbol{\alpha}_2 +\cdots+ k_n\boldsymbol{\alpha}_n = [\boldsymbol{\alpha}_1, \boldsymbol{\alpha}_2,\ldots,\boldsymbol{\alpha}_n] \left[ \begin{matrix}k_1 \\k_2\\ \vdots \\k_n\end{matrix}\right] \]

零向量可以被任何向量组线性表示。

  1. 线性相关

若存在一组不全为零的数 \(k_1,k_2,\ldots,k_n\)，使得向量组 \(\boldsymbol{\alpha}_1,\boldsymbol{\alpha}_2,\ldots,\boldsymbol{\alpha}_n\) 的线性组合

\[ k_1\boldsymbol{\alpha}_1 + k_2\boldsymbol{\alpha}_2 +\cdots+ k_n\boldsymbol{\alpha}_n = 0 \]

则称 \(\boldsymbol{\alpha}_1,\boldsymbol{\alpha}_2,\ldots,\boldsymbol{\alpha}_n\) 『线性相关』。

  1. 线性无关

若不存在一组不全为零的数 \(k_1,k_2,\ldots,k_n\)，使得 \(k_1\boldsymbol{\alpha}_1 + k_2\boldsymbol{\alpha}_2+\cdots+ k_n\boldsymbol{\alpha}_n = 0\) 成立，即当且仅当 \(k_1 = k_2 =\cdots= k_n = 0\) 时，才有 \(k_1\boldsymbol{\alpha}_1 + k_2\boldsymbol{\alpha}_2+\cdots+ k_n\boldsymbol{\alpha}_n = 0\) 成立，则称 \(\boldsymbol{\alpha}_1,\boldsymbol{\alpha}_2,\ldots,\boldsymbol{\alpha}_n\) 『线性无关』或『线性独立』。

一个确定的向量组若不线性相关，则线性无关。反之成立。

  1. 重要结论

  • 向量组线性相关 \(\Leftrightarrow\) 向量组中至少一个向量可以被其他向量线性表示。

    故含有零向量的向量组一定线性相关。

  • 向量组 \(\boldsymbol{\beta}_1,\boldsymbol{\beta}_2, \ldots, \boldsymbol{\beta}_t\) 可由向量组 \(\boldsymbol{\alpha}_1,\boldsymbol{\alpha}_2,\ldots,\boldsymbol{\alpha}_s\) 线性表示，且 \(t > s\)，则 \(\boldsymbol{\beta}_1,\boldsymbol{\beta}_2, \ldots, \boldsymbol{\beta}_t\) 线性相关。

    以少表多，多的相关。

  • 向量组的向量个数大于向量的维数，则向量组线性相关。

  • 向量组 \(\boldsymbol{\alpha}_1,\boldsymbol{\alpha}_2,\ldots,\boldsymbol{\alpha}_n\) 线性无关，\(\boldsymbol{\beta},\boldsymbol{\alpha}_1,\boldsymbol{\alpha}_2,\ldots,\boldsymbol{\alpha}_n\) 线性相关，则 \(\boldsymbol{\beta}\) 可被 \(\boldsymbol{\alpha}_1,\boldsymbol{\alpha}_2,\ldots,\boldsymbol{\alpha}_n\) 线性表示，且表示方法唯一。

    推论：\(\boldsymbol{\beta}\) 可由向量组 \(\boldsymbol{\alpha}_1,\boldsymbol{\alpha}_2,\ldots,\boldsymbol{\alpha}_n\) 线性表示，则表示方法唯一 \(\Leftrightarrow \boldsymbol{\alpha}_1,\boldsymbol{\alpha}_2,\ldots,\boldsymbol{\alpha}_n\) 线性无关。

  • 向量组 \(\boldsymbol{\alpha}_1,\boldsymbol{\alpha}_2,\ldots,\boldsymbol{\alpha}_n\) 线性无关 \(\Leftrightarrow [\boldsymbol{\alpha}_1,\boldsymbol{\alpha}_2,\ldots,\boldsymbol{\alpha}_n]\boldsymbol{x} = \boldsymbol{0}\) 只有零解。\(\boldsymbol{\alpha}_1,\boldsymbol{\alpha}_2,\ldots,\boldsymbol{\alpha}_n\) 线性相关 \(\Leftrightarrow [\boldsymbol{\alpha}_1,\boldsymbol{\alpha}_2,\ldots,\boldsymbol{\alpha}_n]\boldsymbol{x} = \boldsymbol{0}\) 有非零解。

  • 向量 \(\boldsymbol{\beta}\) 可由向量组 \(\boldsymbol{\alpha}_1,\boldsymbol{\alpha}_2,\ldots,\boldsymbol{\alpha}_n\) 线性表示 \(\Leftrightarrow [\boldsymbol{\alpha}_1,\boldsymbol{\alpha}_2,\ldots,\boldsymbol{\alpha}_n]\boldsymbol{x} = \boldsymbol{\beta}\) 有解。\(\boldsymbol{\beta}\) 不可由向量组 \(\boldsymbol{\alpha}_1,\boldsymbol{\alpha}_2,\ldots,\boldsymbol{\alpha}_n\) 线性表示 \(\Leftrightarrow [\boldsymbol{\alpha}_1,\boldsymbol{\alpha}_2,\ldots,\boldsymbol{\alpha}_n]\boldsymbol{x} = \boldsymbol{\beta}\) 无解。

  • 向量组内的部分向量线性相关 \(\Rightarrow\) 整个向量组也线性相关。

    逆否命题：向量组线性无关 \(\Rightarrow\) 向量组内的部分向量也线性无关。

  • 如果一组 n 维向量 \(\boldsymbol{\alpha}_1,\boldsymbol{\alpha}_2, \ldots, \boldsymbol{\alpha}_s\) 线性无关，则在每个向量的相同位置同时添加若干个分量所得的向量组 \(\boldsymbol{\alpha}^+_1,\boldsymbol{\alpha}^+_2, \ldots, \boldsymbol{\alpha}^+_s\) 仍线性无关。如果一组 n 维向量 \(\boldsymbol{\alpha}_1,\boldsymbol{\alpha}_2, \ldots, \boldsymbol{\alpha}_s\) 线性相关，则在每个向量的相同位置同时去掉若干个分量所得的向量组 \(\boldsymbol{\alpha}^-_1,\boldsymbol{\alpha}^-_2, \ldots, \boldsymbol{\alpha}^-_s\) 仍线性相关。

    延长仍无关，缩短仍相关。

  • 初等行变换不改变矩阵的列向量的线性关系，初等列变换不改变矩阵的行向量组的线性关系。

    譬如向量组 \(\boldsymbol{\alpha}_1,\boldsymbol{\alpha}_2,\boldsymbol{\alpha}_3\) 做初等行变换得到 \(\boldsymbol{\beta}_1,\boldsymbol{\beta}_2, \boldsymbol{\beta}_3\)，若 \(k_1\boldsymbol{\alpha}_1 + k_2\boldsymbol{\alpha}_2 = \boldsymbol{\alpha}_3\)，则 \(k_1\boldsymbol{\beta}_1 + k_2\boldsymbol{\beta}_2 = \boldsymbol{\beta}_3\)。

  • 向量组内的向量两两正交 \(\Rightarrow\) 向量组线性无关。

    注：向量组内的向量两两线性无关 \(\not\Rightarrow\) 向量组线性无关。

向量组的秩和等价

向量组的秩等于向量组的『极大线性无关组』的向量个数，即向量组 \(\mathcal{A}\) 内最多找到 \(r(\mathcal{A})\) 个线性无关的向量。

  • 向量组的秩等于向量组张成的向量空间的维数，向量组的极大线性无关组是向量组张成的向量空间的一组基。

  • 向量组 \(\boldsymbol{\alpha}_1,\boldsymbol{\alpha}_2,\ldots ,\boldsymbol{\alpha}_s\) 线性无关 \(\Leftrightarrow r(\boldsymbol{\alpha}_1,\boldsymbol{\alpha}_2,\ldots ,\boldsymbol{\alpha}_s) = s\)。

  • 向量组 \(\mathcal{B}\) 可由向量组 \(\mathcal{A}\) 线性表示 \(\Leftrightarrow r(\mathcal{A}) = r(\mathcal{A},\mathcal{B}) \Rightarrow r(\mathcal{B}) \leq r(\mathcal{A})\)。

    此处的 \(r(\mathcal{B}) \leq r(\mathcal{A})\)，仅当 \(\mathcal{A} \cong \mathcal{B}\) 时等号成立。

  • 向量组 \(\mathcal{A} : \boldsymbol{\alpha}_1,\boldsymbol{\alpha}_2,\ldots ,\boldsymbol{\alpha}_s, \; \mathcal{B}:\boldsymbol{\beta}_1,\boldsymbol{\beta}_2,\ldots,\boldsymbol{\beta}_t\) 中的向量『同维』，若 \(\mathcal{A}\) 中的向量都可以由 \(\mathcal{B}\) 线性表示，且 \(\mathcal{B}\) 中的向量都可以由 \(\mathcal{A}\) 线性表示 \(\Leftrightarrow \mathcal{A} \cong \mathcal{B}\)。

  • 向量组 \(\mathcal{A}:\boldsymbol{\alpha}_1,\boldsymbol{\alpha}_2,\ldots ,\boldsymbol{\alpha}_s,\; \mathcal{B} : \boldsymbol{\beta}_1,\boldsymbol{\beta}_2,\ldots,\boldsymbol{\beta}_t\) 中的向量同维，则 \(\mathcal{A} \cong \mathcal{B}\Leftrightarrow r(\mathcal{A}) = r(\mathcal{B}) = r(\mathcal{A},\mathcal{B})\)。

  • \(\boldsymbol{\alpha}_1,\boldsymbol{\alpha}_2,\ldots ,\boldsymbol{\alpha}_s \cong \boldsymbol{\beta}_1,\boldsymbol{\beta}_2,\ldots,\boldsymbol{\beta}_t \Leftrightarrow \text{span}\{\boldsymbol{\alpha}_1,\boldsymbol{\alpha}_2,\ldots ,\boldsymbol{\alpha}_s\} = \text{span}\{\boldsymbol{\beta}_1,\boldsymbol{\beta}_2,\ldots,\boldsymbol{\beta}_t\}\) 即两向量组张成的空间重合。

  • 设向量组 \(\boldsymbol{\beta}_1,\boldsymbol{\beta}_2,\ldots,\boldsymbol{\beta}_n\) 线性无关并可由向量组 \(\boldsymbol{\alpha}_1,\boldsymbol{\alpha}_2,\ldots ,\boldsymbol{\alpha}_n\)（向量个数相同）线性表示，则两向量组等价。

  • 两矩阵的列（行）向量组等价，两矩阵未必等价（前提是同型）。

  • 设 \(\boldsymbol{R}\) 为行满秩矩阵 \(\Rightarrow \boldsymbol{AR}\) 与 \(\boldsymbol{A}\) 的列向量组等价。

  • 设 \(\boldsymbol{C}\) 是列满秩矩阵 \(\Rightarrow \boldsymbol{CA}\) 与 \(\boldsymbol{A}\) 的行向量组等价。

  • 设 \(\boldsymbol{Q}\) 是可逆矩阵 \(\Rightarrow \boldsymbol{AQ}\) 与 \(\boldsymbol{A}\) 的列向量组等价，\(\boldsymbol{QA}\) 与 \(\boldsymbol{A}\) 的行向量组等价。

  • 特殊情况 1：\(\boldsymbol{A}\boldsymbol{A}^{\text{T}}\) 与 \(\boldsymbol{A}\) 的列向量组等价，\(\boldsymbol{A}^{\text{T}}\boldsymbol{A}\) 与 \(\boldsymbol{A}\) 的行向量组等价。

  • 特殊情况 2：\(\boldsymbol{\alpha}_1,\boldsymbol{\alpha_2},\ldots, \boldsymbol{\alpha_n}\) 线性无关，且 \(\boldsymbol{\alpha}_1,\boldsymbol{\alpha_2},\ldots, \boldsymbol{\alpha_n} \cong \boldsymbol{\beta}_1,\boldsymbol{\beta_2},\ldots, \boldsymbol{\beta_n} \Leftrightarrow\) 存在可逆矩阵 \(\boldsymbol{Q}\)，使得

\[ [\boldsymbol{\alpha}_1,\boldsymbol{\alpha}_2,\ldots ,\boldsymbol{\alpha}_n]\boldsymbol{Q} = [\boldsymbol{\beta}_1,\boldsymbol{\beta}_2,\ldots,\boldsymbol{\beta}_n] \]

  • 寻找向量组的极大无关组的方法：

    • 用矩阵表示列向量组，通过初等行变换化为阶梯形。得出向量组的秩 = 矩阵的秩 = 阶梯数。

      初等行变换不改变列向量间的线性关系。

    • 按列寻找 \(r\) 个线性无关的列向量即可。其中，矩阵的主元列一定是极大无关组。

向量空间和基

  1. 向量运算

对于列向量 \(\boldsymbol{\alpha} = [a_1,a_2,\ldots,a_n]^{\text{T}}\) 和 \(\boldsymbol{\beta} = [b_1,b_2,\ldots,b_n]^{\text{T}}\)。

  • 向量加法：\(\boldsymbol{\alpha} + \boldsymbol{\beta} = [a_1+b_1,a_2+b_2,\ldots,a_n+b_n]^{\text{T}}\)。
  • 数乘：\(k\boldsymbol{\alpha} = [ka_1,ka_2,\ldots,ka_n]^{\text{T}}\)。
  • 标准内积：\(\langle \boldsymbol{\alpha}, \boldsymbol{\beta}\rangle = \boldsymbol{\alpha} \cdot \boldsymbol{\beta} = \boldsymbol{\alpha}^{\text{T}}\boldsymbol{\beta} = \boldsymbol{\beta} ^{\text{T}}\boldsymbol{\alpha} = a_1b_1 + a_2b_2 +\cdots+a_nb_n\)。
  • 向量的模：\(\| \boldsymbol{\alpha} \| = \sqrt{\boldsymbol{\alpha} \cdot \boldsymbol{\alpha}} = \sqrt{\boldsymbol{\alpha}^{\text{T}}\boldsymbol{\alpha}}\)。
  • \(\boldsymbol{\alpha} \perp \boldsymbol{\beta} \Leftrightarrow \boldsymbol{\alpha} \cdot \boldsymbol{\beta} = 0\)。
  1. 向量空间

向量空间的公理化定义：

\(\mathbf{F}\) 是一个域，\(V\) 是一个非空集合，\(V\) 在 \(\mathbf{F}\) 上对两种运算封闭：

  • 向量加法：若 \(\boldsymbol{v,}\boldsymbol{w} \in V\)，则 \(\boldsymbol{v} + \boldsymbol{w} \in V\)。
  • 数乘：若 \(\lambda \in \mathbf{F},\;\boldsymbol{v} \in V\)，则 \(\lambda \boldsymbol{v} \in V\)。

且这两种运算满足向量空间的八条公理，则 \(V\) 是 \(\mathbf{F}\) 上的一个『向量空间』。

简单定义：\(\mathbb{R}^n\) 是所有 n 维实向量的集合，\(\mathbb{R}^n\) 内任意向量组的线性组合仍属于 \(\mathbb{R}^n\)，所以 \(\mathbb{R}^n\) 是一个向量空间。

\(\mathbb{R}^n\) 是最常见的向量空间。如果 \(\mathbb{R}^n\) 不仅定义了向量加法和数乘运算，还定义了向量内积运算，则 \(\mathbb{R}^n\) 是『欧氏空间』。

除 \(\mathbb{R}^n\) 外，还有其他的向量空间，如：

  • \(\mathbb{R}^n\) 的子空间。
  • 只含有零向量的向量空间。

向量空间一定含有零向量。

  1. 子空间

若集合 \(W\) 是向量空间 \(V\) 的子集，且 \(W\) 也是一个向量空间，则称 \(W\) 是 \(V\) 的『子空间』。

  1. 张成

向量空间 \(V\) 中一向量组 \(\boldsymbol{\alpha}_1, \boldsymbol{\alpha}_2,\ldots,\boldsymbol{\alpha}_n\) 的所有线性组合构成的向量集合称为这个向量组的『张成』，记作 \(\text{span} \{\boldsymbol{\alpha}_1,\boldsymbol{\alpha}_2,\ldots,\boldsymbol{\alpha}_n\}\)。

  • \(\text{span} \{\boldsymbol{\alpha}_1,\boldsymbol{\alpha}_2,\ldots,\boldsymbol{\alpha}_n\}\) 一定是 \(V\) 的子空间。
  • \(\boldsymbol{\alpha}_1,\boldsymbol{\alpha}_2,\ldots,\boldsymbol{\alpha}_n\) 的极大线性无关组是 \(\text{span} \{\boldsymbol{\alpha}_1,\boldsymbol{\alpha}_2,\ldots,\boldsymbol{\alpha}_n\}\) 的一组基。
  • 如果 \(\text{span}\{\boldsymbol{\alpha}_1,\boldsymbol{\alpha}_2,\ldots,\boldsymbol{\alpha}_n\} = V\)，即 \(V\) 中的所有向量都能被 \(\boldsymbol{\alpha}_1,\boldsymbol{\alpha}_2,\ldots,\boldsymbol{\alpha}_n\) 线性表示，则称 \(\boldsymbol{\alpha}_1, \boldsymbol{\alpha}_2,\ldots,\boldsymbol{\alpha}_n\) 张成 \(V\)。
  1. 基

若向量空间 \(V\) 中一向量组 \(\mathcal{A} : \boldsymbol{\alpha}_1, \boldsymbol{\alpha}_2,\ldots,\boldsymbol{\alpha}_n\) 满足：

  • 是线性无关的。
  • 张成 \(V\)，即 \(\text{span} \{\mathcal{A}\} = V\)。

则 \(\mathcal{A}\) 是向量空间 \(V\) 的一组『基』。\(V\) 的所有基含有的向量数目相同，表示 \(V\) 的『维数』。

一组基就是向量空间内的一个极大线性无关组，向量空间的维数等于向量空间内任意向量组的秩的最大值，即向量空间内任意线性无关组的最大向量个数。

注：向量空间的维数与向量的维数是不同的概念。仅当向量空间是 \(\mathbb{R}^n\) 时，向量空间的维度等于向量空间中向量的维度等于 n。

\(V\) 上的任意向量 \(\boldsymbol{v}\) 都可以被基 \(\mathcal{A}\) 线性表示：

\[ \boldsymbol{v} = x_1\boldsymbol{\alpha}_1 + x_2\boldsymbol{\alpha}_2 +\cdots+x_n\boldsymbol{\alpha}_n = \left[\boldsymbol{\alpha}_1,\boldsymbol{\alpha}_2,\ldots,\boldsymbol{\alpha}_n\right]\left[\begin{matrix}x_1\\x_2\\ \vdots \\ x_n\end{matrix}\right] \]

其中 \([x_1,x_2,\ldots,x_n]^{\text{T}}\) 称为 \(\boldsymbol{v}\) 在基 \(\mathcal{A}\) 下的『坐标』。

  1. 规范正交基

欧氏空间 \(\mathbb{R}^n\) 的标准基 \(\mathcal{E}: \boldsymbol{e}_1 = \begin{bmatrix}1\\ 0\\ 0 \\ \vdots \\ 0 \end{bmatrix}, \boldsymbol{e}_2 = \begin{bmatrix}0\\ 1\\ 0 \\ \vdots \\ 0 \end{bmatrix}, \ldots, \boldsymbol{e}_n = \begin{bmatrix}0\\ 0\\ 0 \\ \vdots \\ 1 \end{bmatrix}\) 由一组两两正交的单位向量构成。这种标准基具有优良的性质，即在『正交变换』后依然会得到一组两两正交的向量构成的基。

将欧氏空间的标准基推广到所有向量空间，把一组两两正交的单位向量构成的基称为『标准正交基』或『规范正交基』。

任何一个线性无关的向量组都可以通过『施密特正交化』得到一个等价的正交向量组。即向量空间的任何一组基都可以使用施密特正交化得到正交基，再单位化得到规范正交基。

若 \(\boldsymbol{\alpha}_1, \boldsymbol{\alpha}_2,\ldots,\boldsymbol{\alpha}_n\) 是一组基，则：

\[ \begin{aligned} \boldsymbol{\beta} _1 &= \boldsymbol{\alpha}_1 \\ \boldsymbol{\beta}_2 &= \boldsymbol{\alpha}_2 -\frac{\langle\boldsymbol{\alpha}_2,\boldsymbol{\beta}_1\rangle}{\langle\boldsymbol{\beta}_1, \boldsymbol{\beta}_1\rangle}\boldsymbol{\beta}_1 \\ &\dots\\ \boldsymbol{\beta}_n &=\boldsymbol{\alpha}_n - \frac{\langle\boldsymbol{\alpha}_n,\boldsymbol{\beta}_1\rangle}{\langle\boldsymbol{\beta}_1, \boldsymbol{\beta}_1\rangle}\boldsymbol{\beta}_1 -\frac{\langle\boldsymbol{\alpha}_n,\boldsymbol{\beta}_2\rangle}{\langle\boldsymbol{\beta}_2, \boldsymbol{\beta}_2\rangle}\boldsymbol{\beta}_2 - \cdots-\frac{\langle\boldsymbol{\alpha}_n,\boldsymbol{\beta}_{m-1}\rangle}{\langle\boldsymbol{\beta}_{n-1}, \boldsymbol{\beta}_{n-1}\rangle}\boldsymbol{\beta}_{n-1} \end{aligned} \]

则 \(\boldsymbol{\beta}_1,\boldsymbol{\beta}_2,\ldots,\boldsymbol{\beta}_n\) 是一组与 \(\boldsymbol{\alpha}_1, \boldsymbol{\alpha}_2,\ldots,\boldsymbol{\alpha}_n\) 等价的正交基，再令

\[ \boldsymbol{e}_i = \frac{\boldsymbol{\beta}_i}{\|\boldsymbol{\beta}_i\|}\;(i = 1,2,\ldots,n) \]

可得一组与 \(\boldsymbol{\alpha}_1, \boldsymbol{\alpha}_2,\ldots,\boldsymbol{\alpha}_n\) 等价的规范正交基 \(\boldsymbol{e}_1,\boldsymbol{e}_2,\ldots,\boldsymbol{e}_n\)。

注：施密特正交化的对象是同一向量空间的基向量，经过施密特正交化后仍是该向量空间的基。不同特征值对应的特征向量属于不同『特征子空间』，不能使用施密特正交化。

  1. 基变换和坐标变换

设 \(\mathcal{A}: \boldsymbol{\alpha}_1,\boldsymbol{\alpha}_2,\ldots,\boldsymbol{\alpha}_n\) 和 \(\mathcal{B} :\boldsymbol{\beta}_1,\boldsymbol{\beta}_2,\ldots,\boldsymbol{\beta}_n\) 是向量空间 \(V\) 的两组基，且

\[ [\boldsymbol{\beta}_1,\boldsymbol{\beta}_2,\ldots,\boldsymbol{\beta}_n] = [\boldsymbol{\alpha}_1,\boldsymbol{\alpha}_2,\ldots,\boldsymbol{\alpha}_n]\boldsymbol{C} \]

则 \(\boldsymbol{C}\) 称为由基 \(\mathcal{A}\) 到基 \(\mathcal{B}\) 的『过渡矩阵』。

对于 \(V\) 上的任意向量 \(\boldsymbol{v} = \left[\boldsymbol{\alpha}_1,\boldsymbol{\alpha}_2,\ldots,\boldsymbol{\alpha}_n\right]\left[\begin{matrix}x_1\\x_2\\ \vdots \\ x_n\end{matrix}\right] =\left[\boldsymbol{\beta}_1,\boldsymbol{\beta}_2,\ldots,\boldsymbol{\beta}_n\right]\left[\begin{matrix}y_1\\y_2\\ \vdots \\ y_n\end{matrix}\right]\)，故 \(\left[\begin{matrix}x_1\\x_2\\ \vdots \\ x_n\end{matrix}\right] = \boldsymbol{C}\left[\begin{matrix}y_1\\y_2\\ \vdots \\ y_n\end{matrix}\right]\)。

重要结论：

根据向量组的知识可知：过渡矩阵一定是可逆矩阵，且可逆矩阵一定可以将一组基化为另一组基。

  • 即若 \(\boldsymbol{\alpha}_1,\boldsymbol{\alpha}_2,\ldots,\boldsymbol{\alpha}_n\) 是一组基，且

\[ [\boldsymbol{\beta}_1,\boldsymbol{\beta}_2,\ldots,\boldsymbol{\beta}_n] = [\boldsymbol{\alpha}_1,\boldsymbol{\alpha}_2,\ldots,\boldsymbol{\alpha}_n]\boldsymbol{C} \]

则 \(\boldsymbol{\beta}_1,\boldsymbol{\beta}_2,\ldots,\boldsymbol{\beta}_n\) 是一组基 \(\Leftrightarrow \boldsymbol{C}\) 是可逆矩阵。

推论：\(\boldsymbol{\alpha}_1, \boldsymbol{\alpha}_2,\ldots ,\boldsymbol{\alpha}_s\) 是一个基础解系，则 \(\boldsymbol{\beta}_1, \boldsymbol{\beta}_2,\ldots ,\boldsymbol{\beta}_s\) 是一个基础解系 \(\Leftrightarrow\) 存在可逆矩阵 \(\boldsymbol{C}\)，使得 \([\boldsymbol{\alpha}_1, \boldsymbol{\alpha}_2,\ldots ,\boldsymbol{\alpha}_s]\boldsymbol{C} = [\boldsymbol{\beta}_1, \boldsymbol{\beta}_2,\ldots ,\boldsymbol{\beta}_s]\)。

  • 特别地，若 \(\boldsymbol{\alpha}_1,\boldsymbol{\alpha}_2,\ldots,\boldsymbol{\alpha}_n\) 是一组规范正交基，且

\[ [\boldsymbol{\beta}_1,\boldsymbol{\beta}_2,\ldots,\boldsymbol{\beta}_n] = [\boldsymbol{\alpha}_1,\boldsymbol{\alpha}_2,\ldots,\boldsymbol{\alpha}_n]\boldsymbol{C} \]

则 \(\boldsymbol{\beta}_1,\boldsymbol{\beta}_2,\ldots,\boldsymbol{\beta}_n\) 是规范正交基 \(\Leftrightarrow \boldsymbol{C}\) 是正交矩阵。

向量关系与矩阵乘法

此部分内容极度重要，不仅要学会从矩阵乘法推出向量关系，还要学会将向量关系表示为矩阵乘法的形式。

一个 \(m \times n\) 阶矩阵就是 n 个 m 维列向量，当然也可以是 m 个 n 维行向量：

\[ \boldsymbol{A}_{m \times n} = [\boldsymbol{v}_1, \boldsymbol{v}_2, \ldots, \boldsymbol{v}_n] = \left[ \begin{matrix} {\boldsymbol{w}_1}^{\text{T}}\\ {\boldsymbol{w}_2}^{\text{T}}\\ \vdots \\ {\boldsymbol{w}_m}^{\text{T}} \end{matrix} \right] \; (\boldsymbol{v}_{1,2,\ldots,n} \in \mathbb{R}^m,\; \boldsymbol{w}_{1,2,\ldots,m} \in \mathbb{R}^n) \]

矩阵的主元列是线性无关的，且自由列可由主元列线性表示，故矩阵的主元列是矩阵的列向量组的一个极大线性无关组。

故引出矩阵的秩的第二种定义：矩阵的秩 = 矩阵的列向量组的秩 = 矩阵的行向量组的秩 = 矩阵的列（或行）向量组张成的向量空间的维数。

  1. 矩阵乘向量：
  • 矩阵乘列向量：\(\boldsymbol{Ax}\) 表示 \(\boldsymbol{A}\) 的列向量组的线性组合：

\[ \boldsymbol{A} \boldsymbol{x} = [\boldsymbol{\alpha}_1,\boldsymbol{\alpha}_2,\ldots,\boldsymbol{\alpha}_n]\left[ \begin{matrix}x_1 \\x_2\\ \vdots \\x_n\end{matrix}\right] = x_1\boldsymbol{\alpha}_1 + x_2\boldsymbol{\alpha}_2+\cdots+x_n\boldsymbol{\alpha}_n \]

  • 矩阵称列向量：\(\boldsymbol{Ax}\) 表示 \(\boldsymbol{A}\) 的每个行向量与列向量 \(\boldsymbol{x}\) 的内积：

\[ \boldsymbol{A} \boldsymbol{x} = \left[ \begin{matrix} {\boldsymbol{\alpha}_1}^{\text{T}} \\{\boldsymbol{\alpha}_2}^{\text{T}}\\ \vdots \\ {\boldsymbol{\alpha}_m}^{\text{T}} \end{matrix}\right] \boldsymbol{x}= \left[ \begin{matrix} {\boldsymbol{\alpha}_1}^{\text{T}} \boldsymbol{x} \\ {\boldsymbol{\alpha}_2}^{\text{T}}\boldsymbol{x}\\ \vdots \\ {\boldsymbol{\alpha}_m}^{\text{T}}\boldsymbol{x}\end{matrix}\right]= \left[ \begin{matrix} {\boldsymbol{\alpha}_1}\cdot \boldsymbol{x} \\ {\boldsymbol{\alpha}_2} \cdot \boldsymbol{x}\\ \vdots \\ {\boldsymbol{\alpha}_m}\cdot \boldsymbol{x}\end{matrix}\right] \]

  • 行向量乘矩阵：\(\boldsymbol{x}^{\text{T}}\boldsymbol{B}\) 表示 \(\boldsymbol{B}\) 的行向量组的线性组合：

\[ \boldsymbol{x}^{\text{T}}\boldsymbol{B} = [x_1,x_2,\ldots,x_n]\left[ \begin{matrix} {\boldsymbol{\beta}_1}^{\text{T}}\\ {\boldsymbol{\beta}_2}^{\text{T}}\\ \vdots \\ {\boldsymbol{\beta_n}}^{\text{T}} \end{matrix} \right] = x_1 {\boldsymbol{\beta}_1}^{\text{T}} + x_2 {\boldsymbol{\beta}_2}^{\text{T}} + \cdots + x_n {\boldsymbol{\beta}_n}^{\text{T}} \]

  • 行向量乘矩阵：\(\boldsymbol{x}^{\text{T}}\boldsymbol{B}\) 表示 \(\boldsymbol{B}\) 的每个列向量与行向量 \(\boldsymbol{x}^{\text{T}}\) 的内积：

\[ \begin{aligned} \boldsymbol{x}^{\text{T}}\boldsymbol{B} = \boldsymbol{x}^{\text{T}} \left[ \boldsymbol{\beta}_1,\boldsymbol{\beta}_2,\ldots,\boldsymbol{\beta}_s \right] &= \left[ \boldsymbol{x}^{\text{T}} \boldsymbol{\beta}_1,\boldsymbol{x}^{\text{T}}\boldsymbol{\beta}_2,\ldots,\boldsymbol{x}^{\text{T}}\boldsymbol{\beta}_s \right]\\ &= \left[ \boldsymbol{x} \cdot \boldsymbol{\beta}_1,\boldsymbol{x}\cdot \boldsymbol{\beta}_2,\ldots,\boldsymbol{x}\cdot\boldsymbol{\beta}_s \right] \end{aligned} \]

  1. 矩阵乘矩阵
  • \(\boldsymbol{AB}\) 表示矩阵 \(\boldsymbol{A}\) 与 \(\boldsymbol{B}\) 的每个列向量相乘，即转化为矩阵乘列向量的形式：

\[ \boldsymbol{A}_{m \times n}\boldsymbol{B}_{n \times s} = \boldsymbol{A}[\boldsymbol{\beta}_1,\boldsymbol{\beta}_2,\ldots,\boldsymbol{\beta}_s] = [\boldsymbol{A\beta}_1,\boldsymbol{A\beta}_2,\ldots,\boldsymbol{A\beta}_s] \]

可得 \(\boldsymbol{AB}\) 的 s 个列向量都是 \(\boldsymbol{A}\) 的列向量组的线性组合，\(\boldsymbol{AB}\) 的列向量组被 \(\boldsymbol{A}\) 的列向量组线性表示。故有 \(r(\boldsymbol{A}) = r([\boldsymbol{A}, \boldsymbol{AB} ])\)。

  • \(\boldsymbol{AB}\) 也可以表示矩阵 \(\boldsymbol{B}\) 与 \(\boldsymbol{A}\) 的每个行向量相乘，即转化为行向量乘矩阵的形式：

\[ \boldsymbol{A}_{m \times n}\boldsymbol{B}_{n \times s} = \left[ \begin{matrix} {\boldsymbol{\alpha}_1}^{\text{T}}\\ {\boldsymbol{\alpha}_2}^{\text{T}}\\ \vdots \\ {\boldsymbol{\alpha}_m}^{\text{T}} \end{matrix} \right] \boldsymbol{B} = \left[ \begin{matrix} {\boldsymbol{\alpha}_1}^{\text{T}}\boldsymbol{B}\\ {\boldsymbol{\alpha}_2}^{\text{T}}\boldsymbol{B}\\ \vdots \\ {\boldsymbol{\alpha}_m}^{\text{T}}\boldsymbol{B} \end{matrix} \right] \]

可得 \(\boldsymbol{AB}\) 的 m 个行向量都是 \(\boldsymbol{B}\) 的行向量组的线性组合，即 \(\boldsymbol{AB}\) 的行向量组被 \(\boldsymbol{B}\) 的行向量组线性表示，故有 \(r(\boldsymbol{B}) = r\left ( \begin{bmatrix} \boldsymbol{B} \\ \boldsymbol{AB} \end{bmatrix} \right)\)。

  • \(\boldsymbol{AB}\) 表示 \(\boldsymbol{A}\) 的每个行向量与 \(\boldsymbol{B}\) 的每个列向量的内积：

\[ \begin{aligned} \boldsymbol{A}_{m \times n}\boldsymbol{B}_{n \times s} = \left[ \begin{matrix} {\boldsymbol{\alpha}_1}^{\text{T}} \\ {\boldsymbol{\alpha}_2}^{\text{T}}\\ \vdots \\{\boldsymbol{\alpha}_m}^{\text{T}} \end{matrix}\right] [\boldsymbol{\beta}_1,\boldsymbol{\beta}_2,\ldots,\boldsymbol{\beta}_s] &= \left[ \begin{matrix} {\boldsymbol{\alpha}_1}^{\text{T}} \boldsymbol{\beta}_1 &{\boldsymbol{\alpha}_1}^{\text{T}} \boldsymbol{\beta}_2& \cdots &{\boldsymbol{\alpha}_1}^{\text{T}} \boldsymbol{\beta}_s \\ {\boldsymbol{\alpha}_2}^{\text{T}} \boldsymbol{\beta}_1 &{\boldsymbol{\alpha}_2}^{\text{T}} \boldsymbol{\beta}_2& \cdots &{\boldsymbol{\alpha}_2}^{\text{T}} \boldsymbol{\beta}_s\\ \vdots & \vdots & \ddots& \vdots\\ {\boldsymbol{\alpha}_m}^{\text{T}} \boldsymbol{\beta}_1 &{\boldsymbol{\alpha}_m}^{\text{T}} \boldsymbol{\beta}_2& \cdots &{\boldsymbol{\alpha}_m}^{\text{T}} \boldsymbol{\beta}_s \end{matrix} \right] \\ & =\left[ \begin{matrix} {\boldsymbol{\alpha}_1} \cdot \boldsymbol{\beta}_1 &{\boldsymbol{\alpha}_1} \cdot \boldsymbol{\beta}_2& \cdots &{\boldsymbol{\alpha}_1} \cdot\boldsymbol{\beta}_s \\ {\boldsymbol{\alpha}_2} \cdot \boldsymbol{\beta}_1 &{\boldsymbol{\alpha}_2} \cdot \boldsymbol{\beta}_2& \cdots &{\boldsymbol{\alpha}_2} \cdot\boldsymbol{\beta}_s\\ \vdots & \vdots & \ddots& \vdots\\ {\boldsymbol{\alpha}_m} \cdot \boldsymbol{\beta}_1 &{\boldsymbol{\alpha}_m} \cdot \boldsymbol{\beta}_2& \cdots &{\boldsymbol{\alpha}_m}\cdot\boldsymbol{\beta}_s \end{matrix} \right] \end{aligned} \]

线性方程组解的结构

线性方程组可表示为矩阵与列向量相乘的形式：

  • 齐次方程组：\(\boldsymbol{Ax} = \boldsymbol{0}\)。
  • 非齐次方程组：\(\boldsymbol{Ax} = \boldsymbol{b}\)。

\(\boldsymbol{A}\) 为 m 行 n 列的矩阵，可视为 n 个 m 维列向量组。\(\boldsymbol{0}\) 为零向量，\(\boldsymbol{b}\) 为 m 维列向量。\(\boldsymbol{x}\) 为 n 维列向量 \([x_1,x_2,\ldots,x_n]^{\text{T}}\)，若当 \(\boldsymbol{x} = \boldsymbol{v}\) 时方程组成立，则 \(\boldsymbol{v}\) 为方程组的一个『解向量』。

矩阵的零空间

假设向量 \(\boldsymbol{v}_1,\boldsymbol{v}_2\) 是方程组 \(\boldsymbol{Ax} = \boldsymbol{0}\) 的任意两个解向量，k 为任意常数，则必有

\[ \boldsymbol{A}(k\boldsymbol{v}_1) = k\boldsymbol{A}\boldsymbol{v}_1 = 0,\;\boldsymbol{A}(\boldsymbol{v}_1 + \boldsymbol{v}_2) = \boldsymbol{A}\boldsymbol{v}_1 + \boldsymbol{A}\boldsymbol{v}_2 = \boldsymbol{0} \]

即 \(k\boldsymbol{v}_1\) 和 \(\boldsymbol{v}_1+\boldsymbol{v}_2\) 仍是方程组 \(\boldsymbol{Ax} = \boldsymbol{0}\) 的解向量，故方程组 \(\boldsymbol{Ax} = \boldsymbol{0}\) 的解向量的集合是一个向量空间。这个空间称为方程组 \(\boldsymbol{Ax} = \boldsymbol{0}\) 的『解空间』，也称矩阵 \(\boldsymbol{A}\) 的『零空间』，记作 \(N(\boldsymbol{A})\)。\(N(\boldsymbol{A})\) 是 \(\mathbb{R}^n\) 的子空间。即零空间可表示为：

\[ N(\boldsymbol{A}) = \{\boldsymbol{x} \in \mathbb{R}^n \vert \boldsymbol{Ax} = \boldsymbol{0}\} \]

  1. 解的非零性

根据向量关系与矩阵乘法中的描述，\(\boldsymbol{Ax} = \boldsymbol{0}\) 表示 \(\boldsymbol{A}\) 的每个行向量与列向量 \(\boldsymbol{x}\) 的内积都为 0，即列向量 \(\boldsymbol{x}\) 与 \(\boldsymbol{A}\) 的行向量组正交，故 \(N(\boldsymbol{A})\) 是 \(\boldsymbol{A}\) 的行向量张成空间的正交补空间。

  • 如果 \(\boldsymbol{A}\) 列满秩，即 \(\boldsymbol{A}\) 的列向量组线性无关，根据线性无关的定义可知当且仅当 \(\boldsymbol{x}\) 为零向量时 \(\boldsymbol{Ax} = \boldsymbol{0}\) 成立。即方程组 \(\boldsymbol{Ax} = \boldsymbol{0}\) 有唯一（零）解，无非零解。

  • 如果 \(\boldsymbol{A}\) 列不满秩，即 \(r(\boldsymbol{A}) < n\)，则 \(\boldsymbol{A}\) 的列向量组线性相关，根据线性相关的定义，方程组 \(\boldsymbol{Ax} = \boldsymbol{0}\) 必有非零解，假设非零解向量为 \(\boldsymbol{v}\)，即 \(\boldsymbol{v} \in N(\boldsymbol{A})\)，则 \(k\boldsymbol{v} \in N(\boldsymbol{A})\)，k 为任意常数，故方程组 \(\boldsymbol{Ax} = \boldsymbol{0}\) 有无穷多解。

    推论：若 \(\boldsymbol{A}\) 为方阵，则 \(|\boldsymbol{A}| = 0 \Leftrightarrow \boldsymbol{Ax} = \boldsymbol{0}\) 有非零解。

  • 如果 \(m < n\)，即 \(\boldsymbol{A}\) 的列向量个数大于列向量的维数，则 \(\boldsymbol{A}\) 的列向量组必线性相关，则方程组 \(\boldsymbol{Ax} = \boldsymbol{0}\) 必有（无穷）非零解。

  1. 解的结构

  • \(\boldsymbol{A}_{m \times n}\begin{bmatrix} x_1 \\ x_2 \\ \vdots \\ x_n \end{bmatrix} = \boldsymbol{0}\)，\(\boldsymbol{A}\) 含有 \(r(\boldsymbol{A})\) 个主元列，\(n-r(\boldsymbol{A})\) 个自由列。\(\boldsymbol{A}\) 的主元列对应的 \(\boldsymbol{x}\) 中的分量称为『约束未知量』，\(\boldsymbol{A}\) 的自由列对应的 \(\boldsymbol{x}\) 中的分量称为『自由未知量』。自由未知量的值是任意取的，约束未知量的值由自由未知量决定，故 \(N(\boldsymbol{A})\) 的维数取决于自由未知量的个数：\(\text{dim(}N(\boldsymbol{A})) = n -r(\boldsymbol{A})\)。

  • \(N(\boldsymbol{A})\) 的一组基称为方程组 \(\boldsymbol{Ax} = \boldsymbol{0}\) 的『基础解系』，所以基础解系是 \(n - r(\boldsymbol{A})\) 个线性无关的解向量 \(\boldsymbol{\xi}_1, \boldsymbol{\xi}_2, \ldots, \boldsymbol{\xi}_{n -r(\boldsymbol{A})}\)，基础解系的线性组合称为方程组 \(\boldsymbol{Ax} = \boldsymbol{0}\) 的『通解』，记作

\[ \boldsymbol{x}_N = k_1\boldsymbol{\xi}_1 + k_2 \boldsymbol{\xi}_2 + \cdots + k_{n -r(\boldsymbol{A})}\boldsymbol{\xi}_{n -r(\boldsymbol{A})} \]

矩阵的列空间

由 \(\boldsymbol{A}\) 的列向量组张成的向量空间称为矩阵 \(\boldsymbol{A}\) 的『列空间』，记作

\[ \begin{aligned} \boldsymbol{A} &= [\boldsymbol{\alpha}_1,\boldsymbol{\alpha}_2,\ldots,\boldsymbol{\alpha}_n] \\ C(\boldsymbol{A}) &= \text{span}\{\boldsymbol{\alpha}_1,\boldsymbol{\alpha}_2,\ldots,\boldsymbol{\alpha}_n\} \end{aligned} \]

列向量的维数为 m，故 \(C(\boldsymbol{A})\) 是 \(\mathbb{R}^m\) 的子空间。矩阵 \(\boldsymbol{A}\) 的主列是 \(\boldsymbol{A}\) 的列向量组的极大线性无关组，故矩阵 \(\boldsymbol{A}\) 的主列是 \(C(\boldsymbol{A})\) 的一组基，矩阵列空间的维数等于矩阵主列的个数等于矩阵的秩，即 \(\text{dim}(C(\boldsymbol{A})) = r(\boldsymbol{A})\)。

  1. 解的存在性

根据向量关系与矩阵乘法中的描述，\(\boldsymbol{Ax}\) 表示 \(\boldsymbol{A}\) 的列向量组的线性组合，所以方程组 \(\boldsymbol{Ax} = \boldsymbol{b}\) 有解等价于 \(\boldsymbol{b}\) 可以被 \(\boldsymbol{A}\) 的列向量组线性表示，等价于 \(\boldsymbol{b} \in C(\boldsymbol{A})\)。

  • 仅当 \(r(\boldsymbol{A}) = r([\boldsymbol{A},\boldsymbol{b}])\) 时，\(\boldsymbol{A}\) 的列向量组与 \([\boldsymbol{A},\boldsymbol{b}]\) 的列向量组等价 \(\Leftrightarrow C([\boldsymbol{A},\boldsymbol{b}]) =C(\boldsymbol{A}) \Leftrightarrow \boldsymbol{b} \in C(\boldsymbol{A})\)，故非齐次线性方程组 \(\boldsymbol{Ax} = \boldsymbol{b}\) 有解的充分必要条件是：\(r(\boldsymbol{A}) = r([\boldsymbol{A},\boldsymbol{b}])\)。

  • 当 \(\text{dim}(C(\boldsymbol{A})) = m\) 时，\(C(\boldsymbol{A}) = \mathbb{R}^m\)，\(\boldsymbol{b}\) 为 m 维列向量，\(\boldsymbol{b} \in \mathbb{R}^m\)，即 \(\boldsymbol{b} \in C(\boldsymbol{A})\)。故当 \(\text{dim}(C(\boldsymbol{A})) = r(\boldsymbol{A}) = m\) 即 \(\boldsymbol{A}\) 行满秩时，非齐次线性方程组 \(\boldsymbol{Ax} = \boldsymbol{b}\) 必有解。

  • 当 \(r(\boldsymbol{A}) = r([\boldsymbol{A},\boldsymbol{b}]) = n\) 时，因为矩阵 \(\boldsymbol{A}\) 列满秩，故 \(\boldsymbol{A}\) 的列向量组线性无关，根据定理 2，\(\boldsymbol{b}\) 能被 \(\boldsymbol{A}\) 的列向量组线性表示，且表示法唯一，故非齐次线性方程组 \(\boldsymbol{Ax} = \boldsymbol{b}\) 有唯一解。

  • 当 \(r(\boldsymbol{A}) = m = n\) 时，即方阵 \(\boldsymbol{A}\) 满秩，结合上两条可知非齐次线性方程组 \(\boldsymbol{Ax} = \boldsymbol{b}\) 有唯一解。

    推论：

    • \(|\boldsymbol{A}| \neq 0 \Rightarrow \boldsymbol{Ax} = \boldsymbol{b}\) 必有（且唯一）解。
    • 若 \(\boldsymbol{A}\) 为方阵，则 \(|\boldsymbol{A}| = 0 \Leftrightarrow \boldsymbol{Ax} = \boldsymbol{b}\) 无解，\(|\boldsymbol{A}| \neq 0 \Leftrightarrow \boldsymbol{Ax} = \boldsymbol{b}\) 有（且唯一）解。

  • 当 \(r(\boldsymbol{A}) = r([\boldsymbol{A},\boldsymbol{b}]) < n\) 时，\(\boldsymbol{A}\) 含有 \(n -r(\boldsymbol{A})\) 个自由列，则 \(\boldsymbol{x}\) 中含有 \(n -r(\boldsymbol{A})\) 个自由变量，自由变量的取值是任意的，所以非齐次线性方程组 \(\boldsymbol{Ax} = \boldsymbol{b}\) 有无穷解。

  1. 解的结构

  • 易证方程组 \(\boldsymbol{Ax} = \boldsymbol{b}\) 的通解无法构成向量空间，故非齐次方程组没有解空间。

  • 设 \(\boldsymbol{\eta}_1,\boldsymbol{\eta}_2\) 是方程组 \(\boldsymbol{Ax} = \boldsymbol{b}\) 的两个解向量，则

\[ \boldsymbol{A}(\boldsymbol{\eta}_1 - \boldsymbol{\eta}_2) = \boldsymbol{A}\boldsymbol{\eta}_1 - \boldsymbol{A}\boldsymbol{\eta}_2 = \boldsymbol{b} - \boldsymbol{b} = \boldsymbol{0} \]

即 \(\boldsymbol{\eta}_1-\boldsymbol{\eta}_2\) 是方程组 \(\boldsymbol{Ax} = \boldsymbol{0}\) 的解向量。

  • 设 \(\boldsymbol{\xi} \in N(\boldsymbol{A})\)，则

\[ \boldsymbol{A}(\boldsymbol{\eta}_1 + \boldsymbol{\xi}) = \boldsymbol{A}\boldsymbol{\eta}_1 + \boldsymbol{A}\boldsymbol{\xi} = \boldsymbol{b} + \boldsymbol{0} = \boldsymbol{b} \]

即 \(\boldsymbol{\eta}_1 + \boldsymbol{\xi}\) 是方程组 \(\boldsymbol{Ax} = \boldsymbol{b}\) 的解向量。

重要结论：

  • 如果确定方程组 \(\boldsymbol{Ax} = \boldsymbol{b}\) 的一个特解 \(\boldsymbol{\eta}^*\) 并确定方程组 \(\boldsymbol{Ax} = \boldsymbol{0}\) 的通解 \(\boldsymbol{x}_N\)，即可表示出方程组 \(\boldsymbol{Ax} = \boldsymbol{b}\) 的通解 \(\boldsymbol{\eta}^* + \boldsymbol{x}_N\)。
  • 方程组 \(\boldsymbol{Ax} = \boldsymbol{b}\) 最多具有 \(n - r(\boldsymbol{A}) + 1\) 个线性无关的解向量。反过来说，若方程组 \(\boldsymbol{Ax} = \boldsymbol{b}\) 具有 \(t\) 个线性无关的解向量，则 \(N(\boldsymbol{A})\) 中最少含有 \(t-1\) 个线性无关的向量。
  1. 非齐次方程组解的存在性与齐次方程组解的存在性的关系
  • 若 \(\boldsymbol{Ax} = \boldsymbol{b}\) 有唯一解 \(\Rightarrow \boldsymbol{Ax} = \boldsymbol{0}\) 只有零解。（充分不必要）
  • 若 \(\boldsymbol{A}\) 是方阵，则 \(\boldsymbol{Ax} = \boldsymbol{b}\) 有唯一解 \(\Leftrightarrow \boldsymbol{Ax} = \boldsymbol{0}\) 只有零解。（充分必要）
  • 若 \(\boldsymbol{Ax} = \boldsymbol{b}\) 有无穷解 \(\Rightarrow \boldsymbol{Ax} = \boldsymbol{0}\) 有非零解。（充分不必要）

同解和公共解

  1. 同解方程组

  • 设 \(\boldsymbol{A} \in \mathbb{R}^{m \times n},\boldsymbol{B} \in \mathbb{R}^{s \times n}\)，若方程组 \(\boldsymbol{Ax} = \boldsymbol{0}\) 与方程组 \(\boldsymbol{Bx} = \boldsymbol{0}\) 具有完全相同的解，则

    • \(\Leftrightarrow N(\boldsymbol{A}) = N(\boldsymbol{B}) \Leftrightarrow \boldsymbol{A}\) 和 \(\boldsymbol{B}\) 的行向量组张成的空间重合 \(\Leftrightarrow \boldsymbol{A}\) 和 \(\boldsymbol{B}\) 的行向量组等价 \(\Leftrightarrow r(\boldsymbol{A}) = r(\boldsymbol{B}) = r\left(\begin{bmatrix} \boldsymbol{A} \\ \boldsymbol{B} \end{bmatrix} \right)\)。

      也需要掌握其反向使用。

    • \(\boldsymbol{A}\) 和 \(\boldsymbol{B}\) 的列向量间具有相同的线性关系。

  • 设 \(\boldsymbol{A} \in \mathbb{R}^{m \times n},\boldsymbol{B} \in \mathbb{R}^{s \times n}\)，若方程组 \(\boldsymbol{Ax} = \boldsymbol{0}\) 的解均是方程组 \(\boldsymbol{Bx} = \boldsymbol{0}\) 的解

    • \(\Leftrightarrow \boldsymbol{Ax} = \boldsymbol{0}\) 的解是 \(\boldsymbol{Ax} = \boldsymbol{0}\) 与 \(\boldsymbol{Bx} = \boldsymbol{0}\) 的公共解 \(\Leftrightarrow \boldsymbol{Ax} = \begin{bmatrix} \boldsymbol{A} \\ \boldsymbol{B} \end{bmatrix}\boldsymbol{x} = \boldsymbol{0}\) 同解 \(\Leftrightarrow r(\boldsymbol{A}) = r\left( \begin{bmatrix} \boldsymbol{A} \\ \boldsymbol{B} \end{bmatrix} \right) \Leftrightarrow \boldsymbol{B}\) 的行向量组可由 \(\boldsymbol{A}\) 的行向量组线性表示。

    也需要掌握其反向使用，设 \(\boldsymbol{M}\) 为任意矩阵，则 \(\boldsymbol{Ax} = \boldsymbol{0}\) 的解均是\(\boldsymbol{MAx} = \boldsymbol{0}\) 的解。

  • 设 \(\boldsymbol{C}\) 是列满秩矩阵，则 \(\boldsymbol{Ax} = \boldsymbol{0}\) 与 \(\boldsymbol{CAx} = \boldsymbol{0}\) 同解。

  • 设 \(\boldsymbol{P}\) 为可逆矩阵，则

\[ \begin{aligned} & \boldsymbol{Ax} = \boldsymbol{0} \Rightarrow \boldsymbol{PAx} = \boldsymbol{0}, \; \boldsymbol{PAx} = \boldsymbol{0} \Rightarrow \boldsymbol{P}^{-1}\boldsymbol{PAx} = \boldsymbol{Ax} = \boldsymbol{0}\\ & \boldsymbol{Ax} = \boldsymbol{b} \Rightarrow \boldsymbol{PAx} = \boldsymbol{Pb}, \; \boldsymbol{PAx} = \boldsymbol{Pb} \Rightarrow \boldsymbol{P}^{-1}\boldsymbol{PAx} = \boldsymbol{P}^{-1}\boldsymbol{Pb} \Rightarrow \boldsymbol{Ax} = \boldsymbol{b} \end{aligned} \]

即对系数矩阵 \(\boldsymbol{A}\) 做初等行变换不改变方程组 \(\boldsymbol{Ax} = \boldsymbol{0}\) 的解，对增广矩阵 \([\boldsymbol{A},\boldsymbol{b}]\) 做初等行变换不改变方程组 \(\boldsymbol{Ax} = \boldsymbol{b}\) 的解。

右乘可逆矩阵 \(\boldsymbol{Q}\) 不能得到相同的结论，但有时可以更方便的解题：

\[ \boldsymbol{AQQ}^{-1}\boldsymbol{x} = \boldsymbol{Ax} \]

故对 \(\boldsymbol{A}\) 做初等列变换（通常是互换两列），然后对 \(\boldsymbol{x}\) 做相应的初等行变换（互换两行），方程组依然成立。

  • \(\boldsymbol{A}^{\text{T}}\boldsymbol{Ax} = \boldsymbol{0}\) 与 \(\boldsymbol{Ax} = \boldsymbol{0}\) 同解。

  • 设 \(\boldsymbol{A}\) 为 n 阶方阵，则 \(\boldsymbol{A}^n \boldsymbol{x} = \boldsymbol{0}\) 与 \(\boldsymbol{A}^{n+1} \boldsymbol{x} = \boldsymbol{0}\) 同解。

  1. 两个方程组的公共解

  • 设 \(\boldsymbol{A} \in \mathbb{R}^{m \times n},\boldsymbol{B} \in \mathbb{R}^{s \times n}\)，则方程组 \(\boldsymbol{Ax} = \boldsymbol{0}\) 与方程组 \(\boldsymbol{Bx} = \boldsymbol{0}\) 的公共解是 \(\begin{bmatrix} \boldsymbol{A} \\ \boldsymbol{B} \end{bmatrix}\boldsymbol{x} = \boldsymbol{0}\) 的解。

  • 同理，方程组 \(\boldsymbol{Ax} = \boldsymbol{\alpha}\) 与方程组 \(\boldsymbol{Bx} = \boldsymbol{\beta}\) 的公共解是 \(\begin{bmatrix} \boldsymbol{A} \\ \boldsymbol{B} \end{bmatrix}\boldsymbol{x} = \begin{bmatrix} \boldsymbol{\alpha} \\ \boldsymbol{\beta}\end{bmatrix}\) 的解。

  • 求出方程组 \(\boldsymbol{Ax} = \boldsymbol{0}\) 的通解 \(k_1\boldsymbol{\xi}_1 + \cdots + k_f\boldsymbol{\xi}_f\)，代入方程组 \(\boldsymbol{Bx} = \boldsymbol{0}\) 得到 \(k_i\;(i =1,\ldots,f)\) 之间到关系。

  • 方程组 \(\boldsymbol{Ax} = \boldsymbol{0}\) 的基础解系 \(\boldsymbol{\xi}_1,\ldots,\boldsymbol{\xi}_p\)，方程组 \(\boldsymbol{Bx} = \boldsymbol{0}\) 的基础解系 \(\boldsymbol{\eta}_1,\ldots, \boldsymbol{\eta}_q\)，则公共解 \(\boldsymbol{\gamma}\) 满足：\(\boldsymbol{\gamma} = k_1\boldsymbol{\xi}_1 + \cdots + k_p\boldsymbol{\xi}_p = l_1\boldsymbol{\eta}_1 + \cdots + l_q\boldsymbol{\eta}_q\)，故 \(k_1\boldsymbol{\xi}_1 + \cdots + k_p\boldsymbol{\xi}_p - l_1\boldsymbol{\eta}_1 - \cdots - l_q\boldsymbol{\eta}_q = \boldsymbol{0}\)，即

\[ [\boldsymbol{\xi}_1, \ldots,\boldsymbol{\xi}_p,\boldsymbol{\eta}_1,\ldots,\boldsymbol{\eta}_q] \begin{bmatrix} k_1 \\ \vdots \\ k_p \\ l_1 \\ \vdots \\ l_q \end{bmatrix} = \boldsymbol{0} \]

求出该方程组的通解，根据通解中 \(k,l\) 的位置与 \(\boldsymbol{\xi}_1,\ldots,\boldsymbol{\xi}_p\) 或 \(\boldsymbol{\eta}_1,\ldots, \boldsymbol{\eta}_q\) 结合，即可得到公共解的通解。

向量组解题思路

  1. 具体型。题目给出了具体的向量和向量关系，此时的思路：

将向量关系化为矩阵的形式，本质上是求解矩阵方程/线性方程组。

  1. 抽象型。题目没有给出具体向量，只提供了向量关系，此时的思路：

  • 设 \(k_1,k_2,\ldots,k_n\) 使得向量满足题目中线性相关或线性无关的要求，然后推导出结果。

  • 化向量关系为抽象矩阵方程，通过向量组的秩、矩阵的秩等结论推导出结论。

  • 形如 \(\boldsymbol{A\alpha}\) 或 \(f(\boldsymbol{A})\boldsymbol{\alpha}\) 的抽象向量，通过 \(\boldsymbol{A}\) 或 \(f(\boldsymbol{A})\) 的特征向量 \(\boldsymbol{\xi}_i\;(i=1,2,\ldots,n)\) 表示：\(\boldsymbol{\alpha}= a_1\boldsymbol{\xi}_1 + a_2\boldsymbol{\xi}_2 + \cdots + a_n\boldsymbol{\xi}_n\)，设 \(\lambda_i\;(i=1,2,\ldots,n)\) 是特征向量 \(\boldsymbol{\xi}_i \;(i=1,2,\ldots,n)\) 的特征值，则

    \[ \boldsymbol{A\alpha} = \boldsymbol{A}(a_1\boldsymbol{\xi}_1 + a_2\boldsymbol{\xi}_2 + \cdots + a_n\boldsymbol{\xi}_n) = a_1\lambda_1\boldsymbol{\xi}_1 + a_2\lambda_2\boldsymbol{\xi}_2 + \cdots + a_n\lambda_n\boldsymbol{\xi}_n \]
