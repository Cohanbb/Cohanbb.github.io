古典概型

随机占位问题

将 n 个可辨质点随机地分配到 N 个盒子中，则：

  • 每盒可容纳任意多个质点：有 \(N^n\) 种分配方法。
  • 每盒可最多容纳一个质点：有 \(\text{P}_N^n = N(N-1)\cdots(N-n+1)\) 种分配方法。

n 个小球的排列数：\(n!\)。

随机抽样问题

  • 先后有放回取 n 次：有 \(N^n\) 种取法。
  • 先后无放回取 n 次：有 \(\text{P}_N^n = N(N-1)\cdots(N-n+1)\) 种取法。
  • 任取 n 个：有 \(\text{C}_N^n = \frac{\text{P}_N^n}{n!}\) 种取法。

\(\text{P}_N^n\) 表示考虑先后顺序地无放回取 n 个的取法，\(\text{C}^n\) 表示一次性取 n 个的取法，等价于无先后顺序且无放回地取 n 个，故 \(\text{C}_N^n\) 等于 \(\text{P}_N^n\) 除以排列数。

几何概型

若 \(\Omega\) 是一个可度量的几何区域，且样本点落入 \(\Omega\) 的某一个可度量子区域 A 的可能性大小与 A 的几何度量成正比，而与 A 的位置与形状无关，称为『几何概型』。

\[ P(A) = \frac{S(A)}{S(\Omega)} \]

重要公式

基本公式

  • \(A \cup (B \cap C) =( A \cup B) \cap (A \cup C),\; A \cap (B \cup C) = (A \cap B) \cup (A \cap C)\)
  • \(\overline{A \cup B} = \overline{A} \cap \overline{B},\;\overline{A \cap B} = \overline{A} \cup \overline{B}\)
  • \(P(A + B) = P(A \cup B)\)
  • \(P(AB) = P(A \cap B) = P\{A,B\}\)

对立事件的概率

\(P(A) = 1 - P(\overline{A})\)

当求 \(P(A)\) 比较困难，而求 \(P(\overline{A})\) 比较容易时，适合使用此公式。

互斥事件的概率公式

\(A \cap B = \varnothing \Leftrightarrow P(AB) = 0 \Leftrightarrow A,B\) 互斥。

对立事件一定互斥。

  • 若 \(A_1,A_2,\ldots,A_n\) 两两互斥，则

\[ P\left( \bigcup_{i=1}^nA_i\right) = \sum_{i =1}^nP(A_i) \]

  • 分割互斥：\(A \cup B = A \cup \overline{A}B = A\overline{B}\cup B = A\overline{B}\cup AB \cup \overline{A}B\)。转化为互斥事件，然后概率可直接相加。

  • 全集分解：\(B_1,B_2,\ldots,B_n\) 为完备事件组，则 \(A = A\Omega =AB_1 \cup AB_2 \cup\cdots\cup AB_n\)。

    两两互斥且和为 \(\Omega\) 的事件组称为完备事件组。

  • \(P(A - B) = P(A\overline{B}) = P(A) - P(AB)\)
  • \(P(A+B) = P(A \cup \overline{A}B) = P(A) + P(\overline{A}B) = P(A) + P(B) - P(AB)\)

  • \(P(A+B+C) = P(A) + P(B) + P(C) - P(AB) - P(BC) - P(AC) + P(ABC)\)

条件概率

  • \(P(A \vert B) = \frac{P(AB)}{P(B)} \;(P(B) > 0)\)
  • 可得结论：

\[ \begin{aligned} P(AB) &= P(B)P(A \vert B)\;(P(B) > 0) \\ &= P(A)P(B \vert A) \;(P(A) > 0) \\ &= P(A) + P(B) - P(A + B) \\ &= P(A) - P(A\overline{B}) \end{aligned} \]

  • 全概率公式：若 \(A_1,A_2,\ldots,A_n\) 为完备事件组，\(P(A_i) > 0\;(i =1,2,\ldots,n)\)，则

\[ P(B) = \sum_{i=1}^n P(A_i)P(B \vert A_i) \]

  • 贝叶斯公式：

\[ P(A_j \vert B) = \frac{P(A_jB)}{P(B)} = \frac{P(A_j)P(B\vert A_j)}{\sum_{i=1}^nP(A_i)P(B \vert A_i)}\;(j = 1,2,\ldots,n) \]

独立事件的概率

若 \(P(AB) = P(A)P(B) \Leftrightarrow A,B\) 相互独立。对于 n 个事件 \(A_1,A_2,\ldots,A_n\) 个事件，若对于其中任意 k 个事件 \(A_{i_1},A_{i_2},\ldots,A_{i_k}\;(k = 2,\ldots,n)\) 都有

\[ P(A_{i_1}A_{i_2}\cdots A_{i_k}) = P(A_{i_1})P(A_{i_2})\cdots P(A_{i_k}) \]

则称 n 个事件相互独立。

譬如

\[ \begin{align} & P(A_1A_2) = P(A_1)P(A_2) \\ & P(A_2A_3) = P(A_2)P(A_3) \\ & P(A_1A_3) = P(A_1)P(A_3) \\ & P(A_1A_2A_3) = P(A_1)P(A_2)P(A_3) \end{align} \]

可说明 \(A_1,A_2,A_3\) 相互独立，若无 (4) 式只能说明 \(A_1,A_2,A_3\) 两两独立。

独立与互斥没什么关系。

结论：

  • \(A,B\) 相互独立 \(\Leftrightarrow A, \overline{B}\) 相互独立 \(\Leftrightarrow \overline{A}, B\) 相互独立 \(\Leftrightarrow \overline{A}, \overline{B}\) 相互独立。
  • \(A,B,C,D\) 相互独立，则对 \(A,B,C,D\) 不含相同事件作运算得到的新事件也独立。
  • \(P(A) > 0,\;A,B\) 相互独立 \(\Leftrightarrow P(B \vert A) = P(B)\)。
  • \(0 < P(A) < 1,\;A,B\) 相互独立 \(\Leftrightarrow P(B \vert A) = P(B \vert \overline{A}) = P(B)\)。
  • \(P(A) = 0 \; \text{or} \; 1\)，则 \(A\) 与任意事件独立。

  • \(0 <P(A) < 1,\; 0 < P(B) < 1\)，且 \(A,B\) 互斥或存在包含关系，则 \(A,B\) 一定不相互独立。

  • 若 \(A_1,A_2,\ldots,A_n\) 相互独立，则

\[ P(A_1A_2\cdots A_n) = P(A_1)P(A_2)\cdots P(A_n) \]

  • 若 \(A_1,A_2,\ldots,A_n\;(n > 3)\) 相互独立，则

\[ \begin{aligned} P\left( \bigcup_{i=1}^nA_i\right) & = 1 - P\left(\overline{ \bigcup_{i =1}^nA_i}\right) \\ & = 1 - P\left(\bigcap_{i=1}^n\overline{A_i} \right) \\ & = 1 - \prod_{i=1}^nP(\overline{A_i}) \\ & = 1 - \prod_{i=1}^n[1-P(A_i)] \end{aligned} \]

不等关系

  • \(0 \leq P(A) \leq 1\)
  • 若 \(A \subseteq B\)，则 \(P(A) \leq P(B)\)。
  • \(AB \subseteq A \subseteq A + B \Rightarrow P(AB) \leq P(A) \leq P(A + B)\)
  • \(\{\max(X,Y) \leq a\} = \{X \leq a\} \cap \{Y \leq a\}\)
  • \(\{\min(X,Y) \leq a\} = \{X \leq a\} \cup \{Y \leq a\}\)
  • \(\{\max(X,Y) \geq a\} = \{X \geq a\} \cup \{Y \geq a\}\)
  • \(\{\min(X,Y) \geq a\} = \{X \geq a\} \cap \{Y \geq a\}\)
  • \(\{\max(X,Y) \leq a\} \subseteq \{\min(X,Y) \leq a\}\)
  • \(\{\min(X,Y) \geq a\} \subseteq \{\max(X,Y) \geq a\}\)
