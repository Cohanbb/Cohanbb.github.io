基本概念

  1. 随机变量

设随机试验 E 的样本空间为 \(\Omega = \{\omega\}\)，若对于 \(\forall \omega \in \Omega\)，都有唯一实数 \(X(\omega)\) 与之对应，且对任意实数 x，\(\{\omega\vert X(\omega) \leq x, \omega \in \Omega\}\) 是随机事件，则称定义在 \(\Omega\) 上的实数单值函数 \(X(\omega)\) 是『随机变量』，简记作 X。

  1. 分布函数

设 X 是随机变量，x 是任意实数，称函数 \(F(x) = P\{X \leq x\}\;(x \in \mathbb{R})\) 为随机变量 X 的『分布函数』，或称 X 服从 \(F(x)\) 分布，记作 \(X \sim F(x)\)（与概率分布、概率密度的表示方法相同，但含义不同）。

根据随机变量的类型可分为：

  • 离散型随机变量及其分布：

若随机变量 X 只能取有限个或可列个值 \(x_1,x_2,\ldots\)，则称 X 为『离散型随机变量』。称

\[ p_i = P\{X = x_i\}\;(i=1,2,\ldots) \]

为 X 的『分布律（列）』或『概率分布』，记作 \(X \sim p_i\)，常用表格或矩阵的形式表示。\(X \sim p_i\) 的分布函数为

\[ F(x) = \sum_{x_i \leq x} p_i \]

  • 连续型随机变量及其分布：

若随机变量 X 的分布函数是连续的，并可表示为

\[ F(x) = \int_{-\infty}^x f(t) \text{d}t \;(x \in \mathbb{R}) \]

其中 \(f(x)\) 为非负可积函数，则称 X 为『连续型随机变量』，称 \(f(x)\) 为 X 的『概率密度函数』，简称『概率密度』，记作 \(X \sim f(x)\)。

  • 混合型随机变量：既非离散也非连续的随机变量。
  1. 重要结论

  • \(F(x)\) 是分布函数 \(\Leftrightarrow F(x)\) 是单调不减、右连续（连续型随机变量的分布函数左右均连续）、非负、有界的函数，且 \(F(-\infty) = 0,\;F(+\infty) = 1\)。

    根据右连续性质，若 \(F(x)\) 是分段函数，则在分段点 xi 处应使用 \(x \ge x_i\)。

  • \(\{p_i\}\) 是概率分布 \(\Leftrightarrow p_i \ge 0\)，且 \(\sum_i p_i = 1\)。

  • \(f(x)\) 是概率密度函数 \(\Leftrightarrow f(x) \geq 0\)，且 \(\int_{-\infty}^{+\infty} f(x) \text{d}x = 1\)。

一维随机变量的分布

离散型分布

\(X \sim p_i\) ，则

\[ F(x) = \sum_{x_i \leq x} p_i \]

分布函数 \(F(x)\) 与分布律 \(p_i\) 互相唯一确定。\(F(x)\) 是『阶梯型』函数。

常见的离散型分布：

伯努利试验： * 每次试验相对独立。 * 每次试验只有成功和失败两种结果。 * 每次试验成功的概率为 p。

  • 0 - 1 分布：

\(X \sim B(1,p)\)，X 为单次伯努利试验成功的次数。其分布律为 \(X \sim \left( \begin{matrix} 1 &0 \\ p &1-p\end{matrix} \right)\)。

  • 二项分布：

\(X \sim B(n,p)\)，X 为 n 重伯努利试验成功的次数。其分布律为

\[ P\{X = k\} = \text{C}_n^k p^k(1-p)^{n-k} \;(k = 1,2,\ldots,n) \]

注：随着 k 的增大，\(P\{x=k\}\) 先上升到最大值然后下降，取最大值时的 k 是 n 次伯努利试验最可能的成功次数，设其为 \(k'\)。 * 当 \((n+1)p\) 为整数时，\(k' = (n+1)p, (n+1)p-1\)。 * 当 \((n+1)p\) 非整数时，\(k' = [(n+1)p]\)。

  • 几何分布：

\(X \sim G(p)\)，若干次伯努利试验，首次成功即停止，X 为试验次数。其分布律为

\[ P\{X = k\} = p(1-p)^{k-1}\; (k =1,2,\ldots) \]

注：几何分布属于离散型等待分布，具有无记忆性。

  • 超几何分布：

经典的古典概型，N 件商品中有 M 件正品，无放回取 n 次，则取到 k 个正品的概率为

\[ P\{X = k\} = \frac{\text{C}_M^k\text{C}_{N-M}^{n-k}}{\text{C}_N^n} \]

  • 泊松分布：

\(X \sim \pi(\lambda)\)，X 表示某单位时间内随机事件发生的次数，\(\lambda\) 表示单位时间内随机事件的平均发生率。其分布律为

\[ P\{X = k\} = \frac{\text{e}^{-\lambda}\lambda^k}{k!} \;(k =1,2,\ldots) \]

对于二项分布 \(X \sim B(n,p)\)，若 n 很大，p 很小，\(\lambda = np\) 适中时，可用泊松分布近似表示：

\[ \text{C}_n^kp^k(1-p)^{n-k} \approx \frac{\text{e}^{-\lambda}\lambda^k}{k!} \]

当 \(n \geq 20,\; p \leq 0.05\) 时逼近效果较好，当 \(n \geq 100, \; np \leq 10\) 时逼近效果更佳。

连续型分布

\(X \sim f(x)\)，则

\[ F(x) = \int_{-\infty}^x f(t) \text{d}t \;(x \in \mathbb{R}) \]

概率密度函数 \(f(x)\) 可唯一确定分布函数 \(F(x)\)，\(F(x)\) 不可唯一确定 \(f(x)\)。

根据可积的性质，改变 \(f(x)\) 的有限个点，不影响 \(F(x)\)。

常见的连续型函数：

  • 均匀分布：

\(X \sim U(a,b)\)，其概率密度函数为

\[ f(x) = \left\{\begin{array}{ll} \frac{1}{b-a}, &a < x < b \; \text{or} \; a \leq x \leq b\\ 0, & \text{other situations}\end{array}\right. \]

其分布函数为

\[ F(x) = \left\{\begin{array}{ll}0, &x < a \\ \frac{x-a}{b-a}, &a \leq x < b \\ 1, & x \geq b\end{array}\right. \]

  • 指数分布：

\(X \sim E(\lambda)\)，其概率密度函数为

\[ f(x) = \left\{ \begin{array}{ll} \lambda \text{e}^{-\lambda x}, &x \geq 0 \\ 0, &\text{other situations}\end{array} \right. \;(\lambda >0) \]

其分布函数为

\[ F(x) = \left\{ \begin{array}{ll} 1-\text{e}^{-\lambda x}, & x \geq 0 \\ 0,&x < 0 \end{array}\right.\;(\lambda>0) \]

注：指数分布属于连续型等待分布，具有无记忆性：

\[ P\{X \geq t+s\vert X \geq t\} = P\{X \geq s\} \]

  • 正态分布：

\(X \sim N \left(\mu,\sigma^2 \right)\)，其概率密度函数为

\[ f(x) = \frac{1}{\sqrt{2\pi}\sigma}\text{e}^{-\frac{(x-\mu)^2}{2\sigma^2}}, \quad -\infty < x < +\infty \]

其中 \(\mu \in \mathbb{R},\; \sigma >0\)。

当 \(\mu = 0,\; \sigma = 1\) 时的正态分布 \(X \sim N(0,1)\) 称为『标准正态分布』，其概率密度函数记作：

\[ \varphi(x) = \frac{1}{\sqrt{2\pi}}\text{e}^{-\frac{x^2}{2}}, \quad -\infty <x < +\infty \]

其分布函数记作：

\[ \Phi(x) = \int_{-\infty}^x \frac{1}{\sqrt{2\pi}}\text{e}^{-\frac{t^2}{2}} \text{d}t \]

标准正态分布具有以下性质：

\[ \begin{aligned} & \Phi(-x) = 1-\Phi(x) \\ & P\{|x| \leq k\} = 2\Phi(k)-1 \;(k > 0)\\ & P\{|x| > k\} = 2[1-\Phi(k)]\;(k > 0) \end{aligned} \]

设 \(X\sim N(\mu,\sigma^2)\)，则

\[ \frac{X- \mu}{\sigma} \sim N(0,1) \]

故

\[ F(x) = \Phi\left( \frac{x-\mu}{\sigma}\right) \]

则

\[ \begin{aligned} & P\{k_1 \leq x \leq k_2\} = \Phi\left( \frac{k_2-\mu}{\sigma}\right) -\Phi\left( \frac{k_1-\mu}{\sigma}\right) \\ & P\{\mu - \sigma \leq x \leq \mu + \sigma \} = 2\Phi(1) -1 \\ & P\{\mu - k\sigma \leq x \leq \mu + k\sigma \} = 2\Phi(k) -1\;(k > 0) \end{aligned} \]

重要结论：

  • \(X \sim N(\mu,\sigma^2) \Rightarrow kX + C \sim N(k\mu + C, k^2\sigma^2)\;(k \neq 0)\)
  • 独立正态分布的线性组合仍为正态分布。设 \(X_i \sim N \left (\mu_i,{\sigma_i}^2 \right)\;(i=1,2,\ldots,n)\)，且 \(X_1,X_2,\ldots,X_n\) 相互独立，则当 \(k_i\) 不全为 0 时，有

\[ \sum_{i=1}^n k_iX_i + C \sim N\left(\sum_{i=1}^nk_1\mu_i+C,\sum_{i=1}^n{k_i}^2{\sigma}^2\right) \]

混合型分布

设 X 是混合型分布，则使用定义法求分布函数：

\[ F(x) = P\{X \leq x \} \]

概率密度为分段函数的分布

设 \(X \sim f(x)\)，若

\[ f(x) = \left\{ \begin{array}{ll} g_1(x), & a \leq x < b \\ g_2(x), & b \leq x < c \\ 0, & \text{other situations}\end{array} \right. \]

则

\[ F(x) = \left \{ \begin{array}{ll} 0, &x < a \\ \int_a^x g_1(t) \text{d}t, & a\leq x <b \\ \int_a^b g_1(t)\text{d}t+\int_b^x g_2(t)\text{d}t, & b \leq x < c \\ 1, & x \geq c \end{array}\right. \]

分布函数求概率

设 \(X \sim F(x)\)，则

\[ \begin{aligned} & P\{X \leq k\} = F(k) \\ & P\{X < k\} = \lim_{x \to k^-}F(x) = F(k^-)\\ & P\{X = k\} = P\{X \leq k\} - P\{X < k\} = F(k) - F(k^-) \\ & P\{k_1 < X \leq k_2\} = P\{X \leq k_2\} - P\{X \leq k_1\} = F(k_2) - F(k_1) \\ & P\{k_1 < X < k_2\} = P\{X < k_2\} - P\{X \leq k_1\} = F({k_2}^-) - F(k_1) \\ & P\{k_1 \leq X \leq k_2\} = P\{X \leq k_2\} - P\{X < k_1\} = F(k_2) - F({k_1}^-) \\ & P\{k_1 \leq X < k_2\} = P\{X < k_2\} - P\{X < k_1\} = F({k_2}^-) - F({k_1}^-) \end{aligned} \]

若 \(X \sim f(x)\)，则

  • \(P\{X < k\} = P\{X \leq k\}\)
  • 设 I 为任意区间，无论是开区间还是闭区间，都有

\[ P\{X \in I\} = \int_I f(t) \text{d}t \]

一维随机变量函数的分布

离散型 - 离散型

设 \(X \sim p_i = P\{X = x_i\} \;(i =1,2,\ldots),\; Y =g(X)\)，则

\[ Y \sim p_i = P\{Y = y_i\} = P\{Y = g(x_i)\} \;(i =1,2,\ldots) \]

连续型 - 连续型

设 \(X \sim f_x(x), \; Y=g(X)\)，则

  1. 分布函数法

\[ F_Y(y) = P\{Y \leq y\} = P\{g(X) \leq y\} = \int _{g(x) \leq y} f_X(x) \text{d} x \]

若 \(F_Y(y)\) 连续且除有限个点外 \(F'_Y(y)\) 存在且连续，则

\[ f_Y(y) = F'_Y(y) \]

  1. 反函数法

若 \(y = g(x)\) 在 \((a,b)\) 上是关于 x 的严格单调可导函数，则存在可导反函数 \(x = h(y) = g^{-1}(y)\)，则

  • 若 \(g(x)\) 单调增加，则 \(h(y)\) 单调增加，即 \(h'(y) > 0\)

\[ F_Y(y) = P\{Y \leq y\} = P\{g(X) \leq y\} = P\{X \leq h(y)\} = \int _{-\infty}^{h(y)}f_X(x)\text{d} x \]

则 \(f_Y(y) = F'_Y(y) = -h'(y)f_X[h(y)]\)。

  • 若 \(g(x)\) 单调减少，则 \(h(y)\) 单调减少，即 \(h'(y) < 0\)

\[ F_Y(y) = P\{Y \leq y\} = P\{g(X) \leq y\} = P\{X \geq h(y)\} = \int _{h(y)}^{+\infty}f_X(x)\text{d} x \]

则 \(f_Y(y) = F'_Y(y) = h'(y)f_X[h(y)]\)。

故得到公式：

\[ \begin{aligned} & f_Y(y) = \left\{ \begin{array}{ll} |h'(y)|f_Y[h(y)], & \alpha < y < \beta \\ 0, & \text{other situations}\end{array}\right. \\ & \alpha = \min(\lim_{x \to a^+}g(x),\lim_{x \to b^-} g(x)),\; \beta =\max(\lim_{x \to a^+}g(x),\lim_{x \to b^-}g(x)) \end{aligned} \]

连续型 - 离散型

设 \(X \sim f_X(x)\)，而 \(Y = g(X)\) 是离散型的随机变量。则先确定 Y 的可能取值 \(y_i\)，然后计算 \(P\{Y = y_i\}\) 得到 Y 的分布律。
