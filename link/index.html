Awesome: Web Link

Abstract

本文整理了一些对编程和学习非常有用的网站链接。

  • Primary \(\Longrightarrow\) 最重要的网站
  • Learning \(\Longrightarrow\) 学习编程和获取资源网站
  • Useful Sites \(\Longrightarrow\) 一些实用的网站
  • Softwares & Projects \(\Longrightarrow\) 实用的软件和项目
  • Infosecurity \(\Longrightarrow\) 信息安全相关
  • Documentations \(\Longrightarrow\) 非常有用的文档

Primary

  • GitHub

    万能网站，上面有各种开源项目、开源软件、程序框架、包和模块以及学习资源。

    也可以自己用来托管代码、分享项目。

  • Google

    遇到问题就 Google。

  • Stack Overflow

    Google 解决不了的就去 Stack Overflow，解决 99% 的问题。

  • CSDN

    很不错的中文社区，一些博客很有价值（垃圾的博客也很多，存在抄袭现象，且下载居然收费）。

  • 思否

    可理解为中文版 Stack Overflow。

  • 知乎

    虽然不是专业的 CS 网站，但知乎上的大佬也很多。

  • Reddit

    Reddit 里的一些社区很有价值：比如 r/webdev, r/programming, r/cpp, r/cpp_questions 等。

Learning

  • LeetCode

    力扣，必备网站，刷题不解释。

  • 哔哩哔哩

    我在 B 站上大学。

  • 菜鸟教程

    Web 前端、服务端、数据分析等各种语言和工具的教程。

    适合当个技术手册。

  • W3school

    跟上面那个没有本质的区别，当技术手册。

  • TheAlgorithm

    GitHub 上的项目 [TheAlgorithm] 的网站，上面有算法和数据结构、机器学习、区块链等的各种语言的实现代码。

  • Data Structure Visualization

    数据结构和算法的可视化。

  • Yanyan's Wiki

  • 廖雪峰的网站

  • immersivemath.com

    沉浸式学习线性代数。

  • CSAPP 重点解读

  • 码农之家

    上面有很多电子书资源。

Useful Sites

  • 在线 LaTeX 公式编辑器

    倒也不是不会写 \(\LaTeX\) 公式，但是它有图片识别出公式的功能，而且非常准确！

  • Overleaf

    强大的在线 \(\LaTeX\) 编辑器，支持可以多人协同合作。

  • 白描 Web

    图片转文字。

  • 路过图床

    提供图片上传和外链服务，稳定快速。

  • DeepL Web

    强大的翻译功能，可以翻译文件。

  • Grammarly

    提供语法纠正功能。

  • draw.io

    免费开源的绘图网站，支持 \(\LaTeX\)。

  • Z-Library

    搜索电子书。

  • 鸠摩搜书

    搜索电子书。

  • Tampermonkey

    油猴插件，yyds！

  • Extfans

    浏览器插件。

  • Greasy Fork

    浏览器插件。

  • ScriptCat

    浏览器插件。

Softwares & Projects

  • Visual Studio Code

    宇宙第一代码编辑器，插件丰富，功能强大，跨平台（Windows/MacOS/Linux/Web）。

  • Typora

    最强 Markodwn 文本编辑器，可惜收费了。

  • Obsidian

    支持实时预览的 Markdown 编辑器，还有文档反链功能，关键是免费。

  • Pandoc

    文件格式转换神器。
